# Business Journey Catalog

Status: Draft | Last updated: 2025-11-22

This catalog lists illustrative business journeys modelled in JourneyForge. Each entry points to artifacts that live together under:

- `docs/3-reference/examples/business/<journey-id>/`

for that journey’s:
- Journey definition (`<journey-id>.journey.yaml`).
- Per-journey OpenAPI export (`<journey-id>.openapi.yaml`).
- Arazzo workflow (`<journey-id>.arazzo.yaml`).
- Human-friendly documentation page (`<journey-id>.md`).
